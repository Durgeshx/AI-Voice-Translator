# 🎤 AI Voice Translator

A real-time AI-powered voice translation system built using **Python**, **OpenAI Whisper**, **PyAnnote Speaker Diarization**, and **Streamlit**.

The application records speech, identifies different speakers, transcribes conversations, translates speech into another language, stores conversation history, generates PDF reports, and provides analytics through an interactive dashboard.

---

# Features

* 🎙 Real-time microphone recording
* 👥 Automatic speaker diarization
* 📝 High-quality speech transcription using Whisper
* 🌍 Multi-language translation
* 💾 Conversation history using SQLite
* 📄 PDF conversation report generation
* 📥 Transcript download
* 📊 Analytics dashboard
* 📈 Speaker statistics
* 🎨 Modern Streamlit interface

# Installation

Clone the repository

```bash
git clone https://github.com/Durgeshx/AI-Voice-Translator.git
```

Move into the project

```bash
cd AI-Voice-Translator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run final_premium_app.py
```
